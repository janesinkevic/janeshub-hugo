# Прошедшее однократное время

Используется для обозначения действия, которое **случилось однажды**. Для спряжения в этом времени **важно знать спряжение глагола**.

### 1. Первое спряжение (а-глаголы)

| Местоимение | Окончание | `Gyventi` (жить) | `Dirbti` (работать) |
| :--- | :--- | :--- | :--- |
| Aš (я) | -au | gyvenau | dirbau |
| Tu (ты) | -ai | gyvenai | dirbai |
| Jis, Ji (он, она) | -o | gyveno | dirbo |
| Mes (мы) | -ome | gyvenome | dirbome |
| Jūs (вы) | -ote | gyvenote | dirbote |
| Jie, Jos (они) | -o | gyveno | dirbo |

### 2. Второе спряжение (і-глаголы)

Глаголы, заканчивающиеся на `-ėti`.

| Местоимение | Окончание | `Norėti` (хотеть) | `Žiūrėti` (видеть) |
| :--- | :--- | :--- | :--- |
| Aš (я) | -ėjau | norėjau | žiūrėjau |
| Tu (ты) | -ėjai | norėjai | žiūrėjai |
| Jis, Ji (он, она) | -ėjo | norėjo | žiūrėjo |
| Mes (мы) | -ėjome | norėjome | žiūrėjome |
| Jūs (вы) | -ėjote | norėjote | žiūrėjote |
| Jie, Jos (они) | -ėjo | norėjo | žiūrėjo |

### 3. Третье спряжение (о-глаголы)

Глаголы, заканчивающиеся на `-yti`.

| Местоимение | Окончание | `Matyti` (видеть) | `Sakyti` (рассказывать) |
| :--- | :--- | :--- | :--- |
| Aš (я) | -iau | mačiau | sakiau |
| Tu (ты) | -ei | matei | sakei |
| Jis, Ji (он, она) | -ė | matė | sakė |
| Mes (мы) | -ėme | matėme | sakėme |
| Jūs (вы) | -ėte | matėte | sakėte |
| Jie, Jos (они) | -ė | matė | sakė |

### 4. Особые случаи (Прошедшее однократное)

* **Глаголы на `-auti`:** Окончание в прошедшем однократном **-avo**.
    * Пример: `Atostogauti` (быть в отпуске) &rarr; `atostogavo`.
* **Глаголы на `-uoti`:** Окончание в прошедшем однократном **-avo**.
    * Пример: `Dainuoti` (петь) &rarr; `dainavo`.
