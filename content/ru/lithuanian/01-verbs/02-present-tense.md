# Настоящее время (Esamasis laikas)

Для спряжения в настоящем времени **критически важно** знать спряжение глагола.

### 1. Первое спряжение (а-глаголы)

Относится к большинству глаголов, заканчивающихся на `-ti`, а также ко всем исключениям.

| Местоимение | Окончание |`Gyventi` (жить) |`Dirbti` (работать) |
| :--- | :--- | :--- | :--- |
| Aš (я) | -u | gyvenu | dirbu |
| Tu (ты) | -i | gyveni | dirbi |
| Jis, Ji (он, она) | -a | gyvena | dirba |
| Mes (мы) | -ame | gyvename | dirbame |
| Jūs (вы) | -ate | gyvenate | dirbate |
| Jie, Jos (они) | -a | gyvena | dirba |

> **Правило:** В 3-м лице единственного и множественного числа форма глагола **совпадает всегда**.

### 2. Второе спряжение (і-глаголы)

Глаголы, заканчивающиеся на `-ėti`. Исключений практически не бывает.

| Местоимение | Окончание | `Norėti` (хотеть) | `Žiūrėti` (видеть) |
| :--- | :--- | :--- | :--- |
| Aš (я) | -iu | noriu | žiūriu |
| Tu (ты) | -i | nori | žiūri |
| Jis, Ji (он, она) | -i | nori | žiūri |
| Mes (мы) | -ime | norime | žiūrime |
| Jūs (вы) | -ite | norite | žiūrite |
| Jie, Jos (они) | -i | nori | žiūri |

### 3. Третье спряжение (о-глаголы)

Глаголы, заканчивающиеся на `-yti`. Исключений практически не бывает.

| Местоимение | Окончание | `Matyti` (видеть) | `Sakyti` (рассказывать) |
| :--- | :--- | :--- | :--- |
| Aš (я) | -au | matau | sakau |
| Tu (ты) | -ai | matai | sakai |
| Jis, Ji (он, она) | -o | mato | sako |
| Mes (мы) | -ome | matome | sakome |
| Jūs (вы) | -ote | matote | sakote |
| Jie, Jos (они) | -o | mato | sako |

### 4. Особые случаи (Настоящее время)

* **Глаголы на `-auti`:** Окончание в настоящем времени **-auja**.
    * Пример: `Atostogauti` (быть в отпуске) &rarr; `atostogauja`.
* **Глаголы на `-uoti`:** Окончание в настоящем времени **-uoja**.
    * Пример: `Dainuoti` (петь) &rarr; `dainuoja`.
